# Global Development Analysis Dashboard

## Overview
This project analyzes World Bank data to understand global and regional development patterns. It covers data cleaning, feature engineering, database storage, and interactive visualization through a Streamlit dashboard.

The goal is to identify trends and relationships across key development indicators such as economy, health, education, environment, gender, and digital access.

---

## Objectives
- Clean and preprocess raw World Bank data
- Create meaningful composite indicators
- Store processed data in a PostgreSQL database
- Perform exploratory data analysis
- Build an interactive dashboard for data exploration

---

## Dataset
Source: World Bank Open Data

---

## Tools and Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- PostgreSQL, SQLAlchemy
- Streamlit

---

## Features
- Data cleaning and preprocessing pipeline
- Feature engineering (health, education, environment, gender indices)
- SQL-based data storage and querying
- Interactive Streamlit dashboard
- KPI indicators and composite development score
- Multi-country comparison and trend analysis

---

## Dashboard Functionality
- Select countries for comparison
- Choose a focus country for detailed analysis
- Adjust year range dynamically
- View KPI cards for key indicators
- Analyze trends using line charts
- Compare countries using bar charts and scatter plots
- Explore relationships using bubble charts

## Key Insights
- Economic development strongly correlates with health and education outcomes
- Regional disparities exist across all development indicators
- High-income countries often rely less on renewable energy
- Gender and digital access remain uneven across regions
